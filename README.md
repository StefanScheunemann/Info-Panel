# Info-Panel
  
##Features  
###Funktionen
* Datum
* Wecker
* Uhrzeit
* Temperatur
* Wetter
* Termine
* offene Aufgaben
* Newsticker
* Witz des Tages
* Vertretungsplan
  
###Mögliche Features
* Gesichtserkennung
* Spracherkennung
* Luftfeuchtigkeit
* Verbindung mit Musikanlage
  
###Eingabemöglichkeiten
* Sprache
* Maus/Tastatur
* Touchscreen
* Gestensteuerung
  
###Aufbau
* jedes Feature ein Block
* Blöcke dynamisch, frei beweglich
* größe von Blöcken anpassbar
* Ablage für nicht genutzt Funktionen
  
###Design
* 20cmx20cm Bildschirm (IPS-Panel)
* eingefasst in Rahmen
* frei aufstellbar
* falls Akku zu schwach => Stromkabel
* Modell siehe Sketchup
